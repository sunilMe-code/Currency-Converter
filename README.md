💱 Currency Converter

A responsive Currency Converter built using HTML, CSS, and Vanilla JavaScript. This project allows users to convert one currency into another using current exchange rates through API integration.

The application demonstrates how to work with APIs, fetch requests, DOM manipulation, and dynamic data updates to create a practical real-world web application.

This project is ideal for beginners looking to improve their JavaScript skills while learning how to integrate external APIs into frontend projects.

✨ Features

💱 Currency conversion

🌍 Multiple currency support

🔄 Real-time exchange rates

🎨 Clean and modern UI

📱 Fully responsive design

🛠️ Tech Stack

HTML5

CSS3

JavaScript (ES6)
